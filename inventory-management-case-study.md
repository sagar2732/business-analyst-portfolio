*Inventory Management System – Business Analysis Case Study
*Business Problem

A mid-size embroidery manufacturing unit is facing challenges in tracking raw material stock, production usage, and finished goods inventory.
Manual inventory tracking using registers and spreadsheets is causing:

Frequent stock shortages

Over-purchasing of materials

Production delays

Lack of real-time visibility

This results in operational inefficiency and revenue loss.

*Project Objective

The objective of this project is to design a centralized inventory management system that will:

Track raw material stock in real time

Monitor production consumption

Manage finished goods availability

Generate automated stock alerts

Improve decision making using reports

* Stakeholders

Business Owner

Production Manager

Store Manager

Purchase Team

Accounts Team

System Users (Machine Operators)

 *As-Is Process (Current Situation)

Raw material entry done manually

Production consumption not recorded properly

Stock verification done monthly

Purchase decisions based on assumptions

No centralized reporting

* To-Be Process (Future System)

System-based stock entry

Production usage auto deduction

Real-time stock dashboard

Low stock automated alerts

Purchase recommendation report

* Functional Requirements

System must allow adding new inventory items

System must track quantity inwards and outwards

System must provide role-based access

System must generate stock valuation report

System must trigger low stock notifications

* Sample User Stories

 User Story 1
As a Store Manager
I want to update raw material stock
So that inventory records remain accurate

 User Story 2
As a Production Manager
I want system to deduct material automatically
So that manual errors are reduced

 User Story 3
As a Business Owner
I want real-time inventory dashboard
So that I can make faster purchasing decisions

* Business Value

30% reduction in stock wastage

Improved production planning

Better working capital utilization

Increased operational efficienc
